# Web-Budget-Tracker

## Description 

This application is intended to track my withdrawals and deposits with or without a data/internet connection
SO THAT my account balance is accurate when I am traveling 

## packages 
-Express.js
=Node.js
=MongoDB
=Mongoose
=Service workers
